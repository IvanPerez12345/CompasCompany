Pluma Multicolor



Que hace esta pluma?

Esta pluma esta diseñada para poder pintar/colorear con una sola cosa sobre toda la infinidad de colores que esten a tu alcanze, el funcionamiento de esta misma pluma es que tu pones la pluma sobre una superficie que tenga cualquier color, y esta absorvera exactamente el tono de color que estas buscando y podras usar ese color de tinta para lo que quieras; esto es algo que no se ha visto antes y esto puede ahorrar el uso de cosas como las cajas de colores, ya que puedes tener todos los colores en una sola pluma, si escuchaste bien ¡Una sola pluma!, asi que, no esperes mas tiempo y llena tu vida de color con este magnifico invento.

En que te sirve?

Esta pluma, como ya fue mencionado esntes te puede ayudar para poder pintar/escribir del color que quieras, puede ser bueno para gente que le guste dibujar ocasionalmente, y tambien hasta para artistas que se dedican a creas pinturas artisticas, esto captura el color se adapta a ese color para pintar y asi de facil puedes conseguir cambiar de color con esta pluma. Que tiene de especial?

Esta pluma literal, es un camaleon ya que tiene la capacidad de cambiar de color, y esto es muy bueno para poder crear arte, inclusive para escribir; puede hacer tus trabajos mas vivos y llenos de clor, tambien te ayuda para ser mas creativo ya que tienes a tu dispocision cualquier colo que puedas usar y esto libera a tu mente para poder darle uso a eso que puedes hacer y hacelo lo mejor posible.

Beneficios:

  • Reduce la contaminacion.
  • Puede tener cualquier color.
  • Es bueno, barato y bonito.
